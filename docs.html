Introduction

Welcome to Butterfly Cloud! This platform offers unparalleled file sharing and management, tailored for individuals and enterprises alike. With its intuitive interface and powerful backend, Butterfly Cloud makes managing your digital resources effortless.

Did you know? Butterfly Cloud ensures your data is always secure and accessible globally.

Why Choose Butterfly Cloud?

Butterfly Cloud is built with speed, security, and simplicity in mind. Whether you are sharing files with colleagues or hosting critical assets, our platform provides a reliable and user-friendly solution.

Install CLI

Requirements

  • Operating Systems: Windows 10, Mac OSX, or Linux
  • Node.js v12 or higher (for CLI)

Install

For Mac users via Homebrew:

$ brew install bncloud-cli

Using NPM:

$ npm install bncloud-cli

Using Yarn:

$ yarn install bncloud-cli
